## 八、LLM 灵感生成前端接入 ✅

### 技术路径

```
🎲 按钮 → studioApi.getInspiration(typeCode)
       → GET /studio/templates/{typeCode}/inspiration
       → StudioService.generateInspiration()
       → LlmRouter.getChatModel() → 豆包大模型
       → 返回 6 个 JSON 灵感种子
       → 前端渲染 ✨ AI 推荐标签
```

### 前端改动

| 变量/函数 | 说明 |
|:---------|:-----|
| `aiInspirations` ref | 存储 AI 生成的灵感数组 |
| `loadingInspirations` ref | 控制加载态（按钮旋转 + 文案） |
| `rollInspiration()` | 调用 API → 解析 JSON → 填充 textarea + 标签 |
| fallback 逻辑 | API 失败 → 回退本地 `typeExamples` + Toast |
