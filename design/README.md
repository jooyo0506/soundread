# 🎨 UI 设计稿

早期产品设计阶段的 HTML 原型页面，用于快速验证交互逻辑和视觉风格。

## 页面清单

| 文件 | 说明 |
|------|------|
| `home.html` | 首页 |
| `login.html` | 登录/注册 |
| `emotion.html` | 情感合成 |
| `ai-director.html` | AI 导演模式 |
| `podcast.html` | AI 播客 |
| `discover.html` | 发现页 |
| `clone.html` | 声音克隆 |
| `vip.html` | VIP 会员 |

> 这些是静态 HTML 原型，最终产品已使用 Vue 3 + TailwindCSS 实现，见 `web/` 和 `admin/` 目录。
