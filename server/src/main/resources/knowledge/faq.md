# 常见问题（FAQ）

## 配额和次数

**问：我还有多少次创作次数？**  
答：请直接问我"我的剩余次数"，我会实时查询你的账户状态为你回答。

**问：次数用完了怎么办？**  
答：可以升级 VIP 套餐，VIP 用户享有更高额度的创作次数。在"我的"页面即可查看套餐详情和购买入口。

**问：免费用户能用哪些功能？**  
答：免费用户可以体验平台核心功能（TTS 配音、AI 工作坊对话等），但每日创作次数和可用音色有一定限制。

## 创作问题

**问：生成的语音在哪里找？**  
答：合成完成的语音保存在"我的创作库"中，也可以在 AI 工作坊历史草稿中找到。

**问：语音合成失败怎么办？**  
答：可以稍后重试。如多次失败，请检查网络连接或联系客服。

**问：可以下载生成的音频吗？**  
答：可以，在创作库或发现页点击下载按钮即可保存到本地。

## 账户问题

**问：如何修改个人信息？**  
答：进入"我的"页面即可修改头像、昵称等信息。

**问：作品会公开吗？**  
答：默认不公开，只有你主动发布到"发现页"的作品才会被其他用户看到。
